# Governance

## Project Leadership

### Maintainers
- Linacre (@LIN4CRE)

### Decision Making
- **Minor changes:** Single maintainer approval
- **Major changes:** Consensus among maintainers
- **Breaking changes:** Require RFC process and community feedback period (14 days minimum)

## Roles

| Role | Responsibilities | How to Attain |
|------|-----------------|---------------|
| Contributor | Submit PRs, report issues | Open to all |
| Committer | Merge PRs, triage issues | Consistent quality contributions |
| Maintainer | Release management, roadmap | Invitation by existing maintainers |

## Release Process
See [RELEASE.md](RELEASE.md)

## Conflict Resolution
1. Discussion in the relevant issue/PR
2. Escalation to maintainers
3. If unresolved: majority vote among maintainers
