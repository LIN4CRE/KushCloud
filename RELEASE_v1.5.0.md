## 🎉 What's New in v1.5.0

### 🔐 CLOUD SYNC & AUTHENTICATION
- **Google Login**: You can now sign in with your Google account to keep your progress safe.
- **Cross-Device Sync**: Play on your phone, then switch to your browser — your coins, skins, and stats will follow you!
- **Auto-Backup**: Your game is automatically saved to the cloud after every run.
- **Conflict Resolution**: Smart merge logic ensures you never lose progress when switching devices.

### 👤 REFINED PROFILE
- **Account Management**: View your connected account details and sync status directly in your profile.
- **Status Indicators**: Real-time feedback on your connection to the cloud (Synced, Syncing, Offline).

### 🔧 ARCHITECTURAL REFACTOR
- **Stability**: Fixed critical module loading issues that caused "Blank Screen" errors.
- **Performance**: Optimized state updates for faster cloud syncing.

## 🚀 How to Use the Release

### For Players

**Play in Browser** (instant, no install):
```
https://lin4cre.github.io/KushCloud/
```

**Download APK for Android**:
1. Go to: https://github.com/LIN4CRE/KushCloud/releases/tag/v1.5.0
2. Download: `kushcloud-v1.5.0-debug.apk`
3. Install on Android device

### For Developers

**Clone & Build Locally**:
```bash
git clone https://github.com/LIN4CRE/KushCloud.git
cd KushCloud
npm install
npm run dev          # dev server
npm run build        # production build
```
