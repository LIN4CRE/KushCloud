# Roadmap

## Completed (v1.0 → v2.3.0)
- Canvas physics engine, procedural audio
- Unlockable cosmetics, XP/coin progression
- Firebase leaderboards, friend system, global chat
- Shop, missions, achievements, loot crates
- Seasonal events, login streaks
- Practice mode, accessibility features
- Anti-cheat validation, run deduplication
- PWA support (manifest, service worker, install detection)
- Update checker with install-type detection
- CI/CD pipeline (GitHub Actions: lint, test, build, deploy, CodeQL)

## Near-term (v2.4.0)
- Cloud save conflict resolution UI
- Chat rate limiting and message moderation
- Additional game worlds and modifiers
- E2E test suite (Playwright)
- Android release signing in CI (keystore via GitHub Secrets)
- Accessibility audit and ARIA improvements

## Future (v3.0+)
- Push notification support (daily mission reminders)
- Community tournaments and events
- Custom avatar creation
- Web monetization / tip jar
- iOS App Store distribution (Apple Developer signing)
- Localization / i18n support
