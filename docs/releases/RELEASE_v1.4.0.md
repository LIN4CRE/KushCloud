## 🎉 What's New in v1.4.0

### 🎯 THE PERFECT PASS MECHANIC
- **Precision Rewards**: Successfully flying through the exact center of a gap now triggers a **"PERFECT!"** bonus.
- **Score Doubler**: Perfect passes grant **2x score** points and an additional combo point.
- **Visual Feedback**: New high-energy particle effects and screen shake for perfect maneuvers.

### 💎 MYTHIC CONTENT ADDED
- **Skin: Solaris**: The eternal sun spirit (Mythic rarity).
- **Trail: Solar Flare**: A blazing sun trail (Mythic rarity).
- Available now in high-tier loot crates and dust crafting.

### 📊 ENHANCED STATISTICS
- **Lifetime Tracking**: The statistics dashboard now tracks your total number of Perfect Passes.
- **Improved UI**: HUD now displays a live counter for perfect passes during your run.

### 🔧 TECHNICAL IMPROVEMENTS
- **Zero Errors**: Fixed all 13 TypeScript errors and warnings identified in the codebase.
- **Save Migration**: Automated migration of player data to version 1.4.0.
- **Engine Optimization**: Refactored `PowerUpManager` for better performance and reliability.

## 🚀 How to Use the Release

### For Players

**Play in Browser** (instant, no install):
```
https://lin4cre.github.io/KushCloud/
```

**Download APK for Android**:
1. Go to: https://github.com/LIN4CRE/KushCloud/releases/tag/v1.4.0
2. Download: `kushcloud-v1.4.0-debug.apk`
3. Install on Android device

### For Developers

**Clone & Build Locally**:
```bash
git clone https://github.com/LIN4CRE/KushCloud.git
cd KushCloud
npm install
npm run dev          # dev server
npm run build        # production build
```
